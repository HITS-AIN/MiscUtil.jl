# MiscUtil.jl

- `sigmoid` and `invsigmoid`
- `makepositive` and `invmakepositive`
- `transformbetween` and `invtransformbetween`

### Matrix relates

- `makesymmetric` and `makesymmetric!`
- `woodbury_A_plus_UCV`

### Other
- `tasksleep`
- `safewrapper`
